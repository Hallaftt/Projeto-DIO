Surgimento da Internet

(Voltar)

A ARPANET foi lançada em 1969 e conectou inicialmente quatro universidades nos Estados Unidos: UCLA, Stanford, UC Santa Barbara e Utah. A ARPANET tinha o objetivo de permitir a troca descentralizada de informações e recursos entre instituições acadêmicas e de pesquisa. Em 1971, a ARPANET contava com 15 nós conectados. O desenvolvimento dos protocolos TCP/IP foi fundamental para o funcionamento da ARPANET. O TCP (Transmission Control Protocol) proporcionava confiabilidade na transmissão de dados, enquanto o IP (Internet Protocol) cuidava do roteamento e entrega dos pacotes. A ARPANET foi um passo importante no desenvolvimento da internet, estabelecendo as bases para a rede global que temos hoje.

Desenvolvimento do TCP/IP

O TCP/IP é um conjunto de protocolos que estabelece as regras para a comunicação na internet. Ele divide os dados em pacotes e garante que eles sejam enviados e recebidos corretamente. O TCP (Transmission Control Protocol) lida com a divisão, sequenciamento e retransmissão de pacotes, enquanto o IP (Internet Protocol) lida com a entrega dos pacotes aos destinos corretos. O TCP/IP permitiu a interconexão de diferentes redes, independentemente do hardware ou software específico que elas utilizavam.

Expansão além do ambiente militar e acadêmico

À medida que a ARPANET crescia, outras redes foram criadas para conectar instituições acadêmicas e de pesquisa em todo o mundo. Em 1986, a Fundação Nacional de Ciências dos Estados Unidos (NSF) lançou a NSFNET, uma rede que se tornou uma infraestrutura fundamental para a pesquisa e educação. A NSFNET expandiu-se rapidamente e, em 1995, foi totalmente desativada, uma vez que a infraestrutura da internet comercial se tornou predominante.

Surgimento da World Wide Web

A World Wide Web foi inventada por Tim Berners-Lee em 1989, enquanto ele trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear). Berners-Lee desenvolveu o HTML (Hypertext Markup Language) para criar páginas da web e o HTTP para transferir dados entre os servidores e os navegadores. A WWW introduziu o conceito de hipertexto, permitindo que as páginas contivessem links para outras páginas, tornando a navegação e a descoberta de informações mais fáceis para os usuários.

Popularização da Internet

A partir da década de 1990, a internet se tornou mais acessível ao público em geral. A proliferação de provedores de acesso à internet comerciais permitiu que os usuários se conectassem à internet a partir de suas casas por meio de modems discados. O surgimento da Web 2.0, com a participação ativa dos usuários e o compartilhamento de conteúdo, impulsionou ainda mais a popularização da internet.

História do HTML

(Voltar)

Surgimento do HTML

O HTML (Hypertext Markup Language) foi criado por Tim Berners-Lee no CERN (Organização Europeia para a Pesquisa Nuclear) em 1989. Berners-Lee desenvolveu o HTML como uma linguagem de marcação para a criação de documentos hipertexto, que poderiam conter links para outras páginas e recursos.

Objetivos do HTML

O objetivo do HTML era permitir que os pesquisadores do CERN compartilhassem informações de forma mais eficiente e colaborativa. Berners-Lee queria criar uma linguagem simples e acessível para a criação de páginas da web, facilitando a troca de informações na incipiente World Wide Web.

Evolução do HTML

A primeira versão do HTML, conhecida como HTML 2.0, foi lançada em 1995. O HTML 3.2 foi lançado em 1997 e introduziu recursos como tabelas e formulários. O HTML 4.01, lançado em 1999, adicionou recursos de estilo e formatação mais avançados.

Padronização do HTML

O HTML foi padronizado pelo World Wide Web Consortium (W3C), um consórcio internacional de organizações que desenvolve padrões para a web. O HTML5, a versão mais recente do HTML, foi lançado em 2014 e trouxe muitos recursos novos, como vídeo, áudio, gráficos vetoriais e elementos semânticos.

Importância do HTML

O HTML é a espinha dorsal da World Wide Web, sendo a linguagem de marcação utilizada para criar páginas e conteúdo na web. O HTML permite que os desenvolvedores estruturem e organizem o conteúdo, definindo a semântica, formatação e interatividade dos elementos na página.

Integração com outras tecnologias

O HTML trabalha em conjunto com o CSS (Cascading Style Sheets) para definir a aparência e o estilo das páginas da web. Também se integra ao JavaScript para adicionar interatividade e funcionalidades dinâmicas às páginas.

Como funciona o HTML?

(Voltar)

Estrutura básica

O HTML é uma linguagem de marcação composta por elementos e tags. Um documento HTML começa com a tag , que engloba todo o conteúdo da página. O conteúdo é dividido em duas seções principais: <head> (cabeçalho) e <body> (corpo).

Tags e elementos:

As tags são usadas para marcar e identificar os elementos dentro de um documento HTML. Cada tag é definida por (abertura) e (fechamento), onde "tag" é o nome da tag específica. Os elementos podem conter texto, imagens, links e outros elementos HTML.

Atributos

As tags podem ter atributos, que fornecem informações adicionais sobre o elemento. Os atributos são especificados dentro da tag de abertura e têm um nome e um valor, separados por um sinal de igual (=). Exemplo: <img src="imagem.jpg" alt="Descrição da imagem">

Semântica

O HTML fornece elementos com significados semânticos para estruturar o conteúdo da página. Exemplos de elementos semânticos são <header>, <nav>, <article>, <section>, <footer>, entre outros. O uso correto desses elementos ajuda na acessibilidade, indexação pelos motores de busca e na compreensão da estrutura da página.

Aninhamento

Os elementos HTML podem ser aninhados uns dentro dos outros para criar uma hierarquia e estrutura de conteúdo. Um elemento pai pode conter um ou mais elementos filhos, que podem conter outros elementos filhos e assim por diante.

Algumas Tags

(Voltar)

  1. <html>: Define o início do documento HTML.
  2. <head>: Contém informações sobre o documento, como o título, metadados e referências a scripts e estilos.
  3. <body>: Contém o conteúdo principal da página, como texto, imagens, links e outros elementos HTML.
  4. <h1> to <h6>: Cabeçalhos de nível 1 a 6, usados para títulos e subtítulos.
  5. <p>: Define um parágrafo de texto.
  6. <a>: Cria um link para outra página ou recurso.
  7. <img>: Insere uma imagem na página.
  8. <ul> e <ol>: Cria listas não ordenadas (com marcadores) e ordenadas (numeradas), respectivamente.
  9. <li>: Define um item de lista em uma <ul> ou <ol>.
  10. <div>: Cria uma divisão ou contêiner genérico para agrupar elementos e aplicar estilos.
